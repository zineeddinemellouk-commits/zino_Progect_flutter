# Logo Replacement Setup Guide

## Overview

This guide will help you replace the old custom-painted logo with the new Attendance University logo across all Flutter platforms.

## What's Been Done

✓ Created `assets/images/` directory
✓ Updated `pubspec.yaml` to include asset directory
✓ Updated `lib/main.dart` to use image asset instead of CustomPaint
✓ Created `generate_app_icons.py` script for bulk icon generation

## Step-by-Step Setup

### Step 1: Save the Logo Image

1. Download or obtain the Attendance University logo image (PNG format, recommended size: 512x512 or larger)
2. Save it as `attendance_logo.png` in the `assets/images/` directory

**Path:** `c:\Users\lenovo\Documents\memoir\assets\images\attendance_logo.png`

### Step 2: Generate All Icon Sizes (Recommended)

This Python script will automatically resize the logo and save it to all required locations:

```bash
# Install Pillow (if not already installed)
pip install Pillow

# Run the script
python generate_app_icons.py
```

**What it does:**

- Resizes the logo to all required dimensions
- Places icons in the correct platform directories:
  - Android: `android/app/src/main/res/mipmap-*`
  - Web: `web/icons/` and favicon
  - macOS: `macos/Runner/Assets.xcassets/AppIcon.appiconset/`

### Step 3: Windows Icon (Manual)

The script creates PNG files, but Windows needs an `.ico` file. You can convert it using:

**Option A: Online Tools**

- Go to https://convertio.co/png-ico/ or similar
- Upload `assets/images/attendance_logo.png`
- Download the `.ico` file
- Save as `windows/runner/resources/app_icon.ico`

**Option B: ImageMagick (if installed)**

```bash
convert assets/images/attendance_logo.png windows/runner/resources/app_icon.ico
```

### Step 4: iOS Icons (if needed)

The Python script handles this, but if you need manual setup:

1. Use the generated PNG files from the script
2. Update `ios/Runner/Assets.xcassets/AppIcon.appiconset/` with the icon files

### Step 5: Verify and Test

```bash
# Update dependencies
flutter pub get

# Run the app
flutter run

# Check that:
# - Logo appears on login screen (200x200)
# - App icon shows in drawer/system
# - Logo is properly centered and high quality
```

## Troubleshooting

### "Image not found" error

- Verify the file exists at: `assets/images/attendance_logo.png`
- Run `flutter pub get` to refresh assets
- Check that `pubspec.yaml` has the `assets:` section configured

### Icon not updating on device

- Clear app cache: `flutter clean`
- Rebuild the app: `flutter run`
- For iOS/macOS: Delete derived data and rebuild

### Logo quality issues

- Ensure source image is high resolution (512x512 minimum)
- Use PNG format with transparency support
- The script uses high-quality LANCZOS resampling

### Different logo between platforms

- Ensure all icons are generated from the same source image
- Run `generate_app_icons.py` again to ensure consistency

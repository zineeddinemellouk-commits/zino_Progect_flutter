# Quick Reference Card

## 🎯 Core Components

### 1. Model
```dart
DepartmentNotificationModel
  - id, departmentId, type, title, message
  - createdAt, isRead, studentId, studentName
  - justificationId, absenceId, subjectName, photoUrl
```

### 2. Service
```dart
DepartmentNotificationService
  - createJustificationNotification() → String (notificationId)
  - watchNotificationsByDepartment(departmentId) → Stream<List>
  - watchUnreadNotificationCount(departmentId) → Stream<int>
  - markAsRead(notificationId) → Future<void>
  - markAllAsRead(departmentId) → Future<void>
  - getJustificationDetails(justificationId) → Future<Map>
```

### 3. Provider
```dart
DepartmentNotificationProvider
  - notifications: List<DepartmentNotificationModel>
  - unreadCount: int
  - isInitialized: bool
  - error: String?

  - initializeDepartmentNotifications(departmentId)
  - markNotificationAsRead(notificationId)
  - markAllNotificationsAsRead(departmentId)
  - getJustificationDetails(justificationId)
  - reset()
```

## 📱 Usage in UI

### Display Bell Icon with Badge
```dart
Consumer<DepartmentNotificationProvider>(
  builder: (context, provider, _) {
    return Stack(
      children: [
        IconButton(
          icon: Icon(Icons.notifications_none_outlined),
          onPressed: () => Navigator.push(
            context,
            MaterialPageRoute(
              builder: (_) => DepartmentNotificationsPage(
                departmentId: 'main-department',
              ),
            ),
          ),
        ),
        if (provider.unreadCount > 0)
          Positioned(
            right: 8,
            top: 8,
            child: Container(
              padding: EdgeInsets.symmetric(horizontal: 6, vertical: 2),
              decoration: BoxDecoration(
                color: Color(0xFFD92D20),
                borderRadius: BorderRadius.circular(10),
              ),
              child: Text(
                provider.unreadCount > 99 ? '99+' : provider.unreadCount.toString(),
              ),
            ),
          ),
      ],
    );
  },
)
```

## 🔧 Common Operations

### Trigger Notification (After Justification Submission)
```dart
final service = DepartmentNotificationService();
await service.createJustificationNotification(
  departmentId: 'main-department',
  studentId: currentUser.uid,
  studentName: 'Ahmed Mohamed',
  justificationId: justRefId,
  absenceId: absenceId,
  subjectName: 'Mathematics',
);
```

### Mark as Read
```dart
final provider = context.read<DepartmentNotificationProvider>();
await provider.markNotificationAsRead(notificationId);
```
